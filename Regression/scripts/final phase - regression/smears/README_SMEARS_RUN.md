# SMEARS Track Runner (PBS/WBC/BMA)

This track runner orchestrates Phases 1-5 for SMEARS only:
- `pbs`
- `wbc`
- `bma`

No biopsy datasets are included.

## Expected input files
- `/home/dineth/focus_measure/journal/Regression/data/manifest_pbs.csv`
- `/home/dineth/focus_measure/journal/Regression/data/manifest_wbc.csv`
- `/home/dineth/focus_measure/journal/Regression/data/manifest_bma.csv`

Each manifest must contain at least:
- `image_path`
- `defocus_um`

## Troubleshooting checklist
- Missing `defocus_um`:
  - Fix source dataset manifests (`manifest_pbs.csv`, `manifest_wbc.csv`, `manifest_bma.csv`).
- Missing Phase 3 cache index:
  - Ensure Phase 3 cache builder runs successfully and creates:
    `/home/dineth/focus_measure/journal/Regression/data/cache_phase3/smears/cache_index.csv`
- Missing phase scripts:
  - Runner reports exact missing paths and stops.
- Split leakage concerns:
  - Splits are group-safe by `group_id` if present, otherwise parent folder of `image_path`.
- Unexpected track mixing:
  - Use SMEARS runner only; it pins datasets to PBS/WBC/BMA commands.
